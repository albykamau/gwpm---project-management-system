Group Projects Management System

A group project management system for JKUAT students.

GET STARTED

Features:

USERS

Manage unlimited amount of users.

projects

Create projects that are connected to clients. Projects are displayed in an agile format and have special sections to store project based credentials.

tasks

Create an unlimited amount of tasks for any project and assign weights and priority per task.

weights

Each task has a weight number, the bigger the weight the harder the task is to complete. This allows you to keep track on how big a project really is.

project sharing

Share your projects with multiples users and collaborate together on your group projects.
