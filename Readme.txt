to import the database to xampp, first create a db in phpmyadmin called 'gwpm' 
then import the file in /GWPM/db/whatever_file_is_in_there.sql

current port: 8000
edit the 'server_start.bat' file to change the port number

run xampp, apache, phpmyadmin and server first
type "localhost:8000" (without the quotes) in your browser to run they system.

passwords are unencrypted for development purposes!
